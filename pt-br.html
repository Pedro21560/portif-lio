Carreira

Especialização Independente em Ciência de Dados e Machine Learning

2024 – Presente

Tenho direcionado meus estudos para a área de dados, com o objetivo de construir uma carreira sólida em Ciência de Dados ou Machine Learning, especialmente voltada ao mercado financeiro.

Tenho grande afinidade com temas como renda variável, ações, criptomoedas e análise quantitativa, e busco aplicar a tecnologia para gerar inteligência de mercado.

Atualmente, aprofundo minhas habilidades práticas em:

  • Programação em Python aplicada à análise de dados
  • Estatística e fundamentos matemáticos para modelagem preditiva
  • SQL e manipulação de dados estruturados
  • Visualização interativa e construção de dashboards analíticos
  • Automatização de relatórios e integração com APIs financeiras

Meus projetos pessoais envolvem a análise de ativos reais, comparação de estratégias de investimento e desenvolvimento de ferramentas preditivas para apoiar decisões no universo da renda variável.
